Explainer: Get closing price data along with time period returns. The N day return for day X is: (X*100)/(X-N) - 100 The weights are used to weight the set of N day returns in the "Index Calc" column. So if you put .2 for 30 days, .4 for 60 days, .1 for 90 days, and .3 for 180 days, then Index calc = .2*30_day_return + .4*60_day_return + .1*90_day_return .3*180_day_return If sum of weights is less than one, leftover ratio goes to 360_day_return

Parameters

filename

symbols -- enter as comma separated symbols without spaces. Ex: VTI,VO,VB,IWC

start_date (YYYY-MM-DD)

end_date (YYYY-MM-DD)

thirty_day_weight (two decimal places)

sixty_day_weight

ninety_day_weight

one_eighty_day_weight
